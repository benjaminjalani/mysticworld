# MysticWorld Tourism Management System

A Django-based web application developed for managing theme park rides and online bookings.

## Features
- User Registration & Login
- Ride Booking System
- Zone-wise Ride Browsing
- Booking Management
- Review & Rating System
- Admin Dashboard

## Technologies Used
- Django
- Python
- SQLite
- HTML, CSS, JavaScript

## Run Project

```bash
python manage.py runserver
```
